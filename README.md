# Hakon

> Hakon for css developer

## Development

``` bash
# frontend
# install dependencies
yarn

# serve with hot reload at localhost:8080
yarn run dev

# tune up election
yarn start

# backend
# install dependencies
yarn

# start server
yarn start
```

## Package

``` bash
# build for production with minification
yarn run build

# build for production and view the bundle analyzer report
yarn run build --report

# package
yarn run package
```

## Test

``` bash
# run unit tests
yarn run unit

# run all tests
yarn test
```
